# raspi3-monitor
Generate simple web graphs for Raspberry Pi's visual monitoring through shell scripts

Installation
-----------------
```shell
apt-get update
apt-get install git -y
git clone https://github.com/ljbitzki/raspi3-monitor.git
cd raspi3-monitor
chmod +x install.sh
/install.sh
```

After instalation, reboot the Pi.
